# PMPlayer
## 基于Windows的自用视频播放器

目前仅支持了MP4格式的视频播放。

快速使用

* 把文件夹拖进程序，可以遍历整个目录下的所有视频，并将目录树结构在程序右边显示。
* 点击具体某个视频播放，有进度条和控制台

快捷键
* 键盘左、右键：快进或快退5秒
